# Sky Bazar

Sky Bazar is a hyperlocal e-commerce app that connects local vendors with nearby customers for fast and convenient shopping.

## 🚀 Features
- Simple user interface
- Interactive button using JavaScript
- Basic frontend structure

## 🛠️ Tech Stack
- HTML
- JavaScript

## 📦 Demo
Open the index.html file in a browser and click the button to see interaction.

## 🔮 Future Plans
- Convert to React-based application
- Add product listings and categories
- Integrate AI chatbot
- Smart recommendations
